Miscellaneous tools for migrating from Plex to Jellyfin

## Setup

Install via [pipx](https://pipx.pypa.io/stable/):

```bash
pipx install jellyplex
```

Or use it via Docker:

```bash
docker run --tty --rm ghcr.io/janw/jellyplex --help
```
