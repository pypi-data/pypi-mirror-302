# Finished Booting

## Summary

Skill to speak when OVOS is ready

## Description

When OpenVoiceOS is started or restarted, this skill will speak a notification to the
user. This speech can be enabled/disabled by intent.

## Examples

- "Enable ready notifications."
- "Disable load speech."
